# Simple Text Editor (Java)

A console-based text editor built using Java and OOP. It uses StringBuffer to perform operations like add, insert, delete, replace, reverse, and clear text through a menu-driven interface.

## Features
- Add text
- Insert text
- Delete text
- Replace text
- Reverse text
- Clear text
- Display text

## How to Run
1. Compile:
   javac Main.java

2. Run:
   java Main
